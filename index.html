Pemrograman Web 1

Media Pembelajaran Perkuliahan

Pemrograman Web 1 adalah mata kuliah dasar yang membahas konsep dan teknik pembuatan website statis. Materi utamanya meliputi pengenalan internet dan web, struktur dasar HTML, penggunaan CSS untuk mengatur tampilan halaman, serta prinsip dasar desain web agar tampilan informatif dan mudah digunakan. Mata kuliah ini bertujuan membekali mahasiswa dengan kemampuan membuat dan menyusun halaman web sederhana secara terstruktur dan konsisten.
